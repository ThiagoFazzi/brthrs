<h1>BRTHRS - Star Wars - Assignment</h1>

# Instructions:

1 - Clone project

- cd brthrs

2 - Backend starter

- cd server
- yarn install
- yarn start

3 - Frontend starter

- cd client
- yarn install
- yarn serve

# Technologies applied on Backend(Server):

- NodeJs
- ExpressJs
- Axios
- Cors

# Technologies applied on Frontend(Client):

- VueJS and Vue CLI
- Axios
